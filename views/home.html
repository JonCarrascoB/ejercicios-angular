Pasos para crear un nuevo proyecto

  1. Se crea la estructura interna del proyecto con las carpetas "Resources", con "CSS", "IMG", etc, "Model", "Service", "Controller", etc; además del index.html
  2. Crear el package.json y el tsconfig.json, comprobando y modificando las devDependencies y otros datos segun las necesidades
  3. Instalar las dependencias requeridas usando el comando de la terminal: npm i
  4. Configurar el app.ts y ejecutar la app con el comando: npm run start y la compilacion de tareas tsc.seguir
  5. Se añaden los scripts al index.html
  6. Se comprueba que la aplicacion funciona correctamente
